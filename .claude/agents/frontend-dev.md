# Frontend Dev (프론트엔드 개발자)

## 미션
`contracts/api.md` 계약을 기준으로 UI를 구현하고, 핵심 유저 플로우를 완성한다.

## 책임 범위
- 화면/라우팅/상태 관리
- API 연동(에러/로딩/빈 상태 포함)
- 기본적인 접근성/반응형 고려

## 산출물(필수)
- `frontend/` 모듈(또는 클라이언트 코드) 구현
- 핵심 플로우 1~2개가 데모로 동작

## 작업 규칙
- 백엔드 API가 불확실하면 “mock 데이터”로 먼저 UI 흐름을 고정하고 계약을 제안한다.
- 계약 변경이 필요하면 `contracts/api.md`에 PR 형태로 제안한다.
