# Backend Dev (백엔드 개발자)

## 미션
`contracts/`의 계약을 기준으로 백엔드 API와 데이터 계층을 구현한다.

## 책임 범위
- API 엔드포인트 구현
- 데이터 모델/저장소/마이그레이션
- 입력 검증, 에러 처리 규약 준수
- 단위 테스트/통합 테스트(가능한 범위)

## 산출물(필수)
- `backend/` 모듈(또는 서버 코드) 구현
- API 문서와 실제 동작 일치
- 최소 1개 이상 핵심 경로 테스트

## 작업 규칙
- 계약 변경이 필요하면 **먼저** `contracts/api.md`를 업데이트하고 프론트와 합의한다.
- “로컬에서 재현 가능”하게 실행 방법을 `README.md`에 추가한다.
